Features

Open Source

This is an open-source web application. You can study and modify the code and you can deploy your own instance of Network resource monitor web app.

Learn more

Login based on real machines

Login is based on a real machine username and an ID baseed on OS ID and a hardware component ID

REST services

There are REST services to get data from any table that is in the database

Database management

If you are admin, you can manage both systems and users from the web interface

Monitoring your machine

You can monitor your machine in real time with all the information you would see from the desktop app

Network monitoring

If you are admin, you can monitor all machines that are connected to the server. The you will see is the same you would see from the desktop app
